# PawLog Docs

User-facing help documentation for the [PawLog](https://github.com/YOUR_USERNAME/PawLog) app — a .NET MAUI pet care companion for Android.

---

## About PawLog

PawLog helps you stay on top of everything your pet needs every day. Daily care routines, medications, weight tracking, vet visits, walk tracking, streaks, achievements, and emergency readiness — all in one place, stored privately on your device.
